## Контекст (бизнес-логика раздела Заказы)

АГР ПАРТС работает с европейскими поставщиками: размещает заказы по проформам, ведёт переписку,
проводит сверку по инвойсу (физически на складе EU), оформляет Контрольный лист (физическая приёмка)
и переводит заказ в «Завершённые» когда КЛ закрыт.

**Трёхуровневый процесс сверки:**

```
Сверка 1 — «Обновить статусы»     → сразу после размещения, статусы по позициям
Сверка 2 — «Сверка заказа»        → финальный инвойс + физический осмотр на складе EU
Контрольный лист                   → отдельный модуль, физическая приёмка
```

```js
// Завершённые заказы

// Переписка (placed-order)
poToggleMsg(header)    // expand/collapse письмо (rotate .po-msg-arrow)
poComposeOpen()        // открыть compose
poComposeClose()       // закрыть compose
poComposeAttach(input) // показать имя прикреплённого файла

poExport()             // CSV с BOM + semicolons → PRF-2026-089_positions.csv

poMoveToCompleted()    // confirm() → тост → nav('orders')

// Сверка
reconcileConfirm()     // закрыть reconcile + тост с деталями многозаказного инвойса
```

## Бизнес-логика: перевод заказа в «Завершённые»

**Условие:** КЛ (Контрольный лист) заказа должен иметь статус «Завершён».  
**Кто переводит:** Оператор вручную, кнопка «В завершённые» в шапке `page-placed-order`.  
**Кнопка заблокирована** (disabled + иконка замка) пока КЛ не завершён.  
**Автоматически заказ не переносится никогда.**

```
КЛ «В работе»  → кнопка disabled (🔒)
КЛ «Завершён»  → кнопка активна → оператор кликает → confirm() → toast → nav('orders')
```

---

## Бизнес-логика: многозаказный инвойс

Один инвойс поставщика может покрывать несколько проформ (заказ на 5 шт из PRF-089 + 2 шт из PRF-088 = итоговый инвойс на 7 шт).

**Сопоставление позиций:**
- Артикул + количество → автоматическое распределение по заказам
- Одинаковое количество в двух заказах → флаг для ручного разбора оператором

**Дубль-артикул (RE522878):**
- PRF-089: 295 шт → PRF-088: 210 шт
- В инвойсе: 505 шт по артикулу RE522878
- Система автоматически разбивает 505 → 295 + 210 по заказным количествам
